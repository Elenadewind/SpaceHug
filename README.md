# 🌌 SpaceHug  
**Отправляйте объятия через космос!**  Идея.

## 🚀 О проекте  
Проект для передачи эмоций через API и AR-интерфейсы.  
- Кастомные HTTP-статусы (218, 520, 666)  
- Интеграция с сервисами доставки  
- AR-визуализация через спутники  

## 🛠️ Технологии  
- Python (FastAPI)  
- React.js + Three.js  
- ARCore/ARKit  

## 📦 Установка  
```bash
git clone https://github.com/Elenadewind/SpaceHug.git
npm install
npm start
